### Run IDBac from R

If you would rather run IDBac directly from R:

```{r}
# install.packages("devtools") # Run this if you don't have the devtools package
devtools::install_github("chasemc/IDBacApp")

# And, to run the app:
IDBacApp::run_app()

```
